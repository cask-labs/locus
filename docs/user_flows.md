# User Flows

This document indexes the core user journeys for the Locus application, mapping the interaction from initialization to daily usage and data visualization.

*   [Onboarding (The Bootstrap)](user_flows/onboarding.md) - Transition from a fresh install to a fully provisioned, cloud-connected system.
*   [Daily Operation (Passive)](user_flows/daily_operation.md) - Continuous, reliable data collection with minimal user intervention.
*   [Visualization (History View)](user_flows/visualization.md) - Verify and explore historical movements.
*   [Manual Sync & Status](user_flows/manual_sync.md) - Immediate verification of data safety.
*   [System Recovery (Re-provisioning)](user_flows/system_recovery.md) - Restore access to existing data on a new device.
